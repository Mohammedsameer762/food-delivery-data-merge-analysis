# Food Delivery Data Merge and Analysis

This project shows how three different data files can be combined into one final dataset for analysis.  
The files include order data, user details, and restaurant information.

The main goal of this project is to create a single dataset that can be used to study sales, users, and restaurant performance.

## Files Used
- orders.csv – contains order and transaction details  
- users.json – contains user information and membership type  
- restaurants.sql – contains restaurant details like cuisine, city, and rating  

## Final Output
- final_food_delivery_dataset.csv – merged dataset used for all analysis

## What This Project Does
- Loads CSV, JSON, and SQL files
- Merges them using user_id and restaurant_id
- Handles missing values
- Creates one clean dataset
- Answers business questions using Python

## Tools Used
- Python
- Pandas
- SQLite
- Google Colab / Jupyter Notebook

## How to Run
1. Open the notebook or Python file  
2. Upload all three data files  
3. Run the cells step by step  
4. The final dataset will be created automatically
